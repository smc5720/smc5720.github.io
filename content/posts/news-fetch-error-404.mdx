> **자동화 알림:** 원문 데이터를 가져오는 단계에서 `Dev.to API 오류: 404`가 발생했습니다. 본문 내용이 전달되지 않아 기사를 작성할 수 없습니다.

## 무슨 일이 있었나

뉴스 포스트 자동 생성 파이프라인이 원문 본문을 받지 못했습니다. 전달된 article 데이터는 다음 한 줄뿐입니다.

```
Dev.to API 오류: 404
```

원문 사실이 없는 상태에서 기사를 지어내는 것은 작성 규칙(사실 외 추측·단정 금지)에 어긋나므로, 본문을 생성하지 않았습니다.

## 확인이 필요합니다

- 대상 article ID가 삭제되었거나 비공개로 전환되었는지 확인하세요.
- `node scripts/fetch-news.mjs` 로 현재 유효한 글 목록을 다시 조회한 뒤, 유효한 ID로 `--id <ID>` 를 재실행하세요.
- 이 `.mdx` 파일은 `draft: true` 로 표시되어 있으며 게시 대상이 아닙니다. 확인 후 삭제하세요.
